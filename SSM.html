Project Details

Project Description

This Android app's UI/UX is redesigned from scratch and developed as well. It is an open-source contribution to GitHub. The application is used to manage lecture timetables, project deadlines, and exam schedules for students. More than 15 active users are using this application, including three college professors.

Java 90%
XML 85%
SQLite 80%
UI/UX Design 75%

Development Process

The development process for this project involved several key steps:

  1. Research & Planning: Understanding user needs and defining project requirements.
  2. Design: Creating wireframes and mockups, focusing on user-centric design principles.
  3. Development: Implementing the design using Java, XML, and SQLite.
  4. Testing: Conducting usability tests and optimizing for different Android devices and versions.
  5. Deployment: Deploying the app to the Google Play Store and monitoring user feedback.

Key Features

Timetable Management

Allows students to manage and organize their lecture schedules efficiently.

Project Deadlines

Helps students keep track of project deadlines and submit work on time.

Exam Schedules

Provides a clear overview of upcoming exams and helps in planning study time.
